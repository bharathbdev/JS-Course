Box Model Parts

Padding (Space inside the element)

This div has padding: 20px (ALL FOUR SIDES)

Code: padding: 20px;

Different Padding Syntaxes

1. One Value - All Sides

padding: 20px; (20px on ALL four sides)

2. Two Values - Vertical | Horizontal

padding: 10px 30px; (10px top/bottom, 30px left/right)

3. Three Values - Top | Left/Right | Bottom

padding: 5px 20px 15px; (5px top, 20px left/right, 15px bottom)

4. Four Values - Top Right Bottom Left (Clockwise)

padding: 5px 10px 15px 25px; (5px top, 10px right, 15px bottom, 25px left)

5. Individual Sides

Individual: padding-top: 20px; padding-right: 10px; padding-bottom: 5px; padding-left: 30px;

Margin (Space outside the element)

This div has margin: 20px

Code: margin: 20px;

Border (Line around the element)

This div has border: 3px solid red

Code: border: 3px solid red;

Combined: Padding + Margin + Border

This div has all three: padding, margin, and border combined

Code: padding: 15px; margin: 10px; border: 2px solid blue;
